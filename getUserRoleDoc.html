Title

 Get User Role API

Function

 To get information of a user as per thier role.

Url

 /getUserRole.php?roles=:roles

Method

 GET

Url params

 Required:
roles=['Mentor','Mentee','Administrator']
example: roles=Mentor

SUCCESS Response

 [{"Emailid":"adisaps8@gmail.com","Status":"Inactive","HashTags":"hired","Roles":"Mentor","TechnologyRatings":"2","PeopleRatings":"1","is_deleted":null,"deletedData":null}]

ERROR Respnse

 Invalid Role
