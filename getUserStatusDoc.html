Title

 Get User Status API

Function

 To get information of a user from its status

Url

 /getUserStatus.php?status=:status

Method

 GET

Url params

 Required:
status=['active','Inactive','Delete','New']
example: status=new

SUCCESS Response

 [{"Emailid":"guptasamanta@yahoo.com","Status":"New","HashTags":null,"Roles":null,"TechnologyRatings":null,"PeopleRatings":null,"is_deleted":null,"deletedData":null}]

ERROR Respnse

 Invalid Status
